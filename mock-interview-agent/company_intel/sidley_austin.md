## System Design Round

- **Format:** Collaborative whiteboarding with interviewer over screen share
- **Tool:** Excalidraw (recommended); candidate's choice of whiteboarding tool
- **Focus:** Web application architecture with emphasis on **Azure-based cloud** expertise
- **No AI tools permitted** unless explicitly approved by the hiring manager during the interview
- Interviewer will probe architectural decisions in real time — explain decisions clearly and in a structured way

**Key Azure services to be fluent in:**
- Compute: Azure App Service, AKS, Azure Functions
- Data: Azure SQL, Cosmos DB, Azure Blob Storage, Azure Data Lake
- Messaging: Azure Service Bus, Event Hubs, Event Grid
- API layer: Azure API Management
- Security / Identity: Azure AD, Managed Identity, Key Vault
- Networking: VNet, Private Endpoints, Application Gateway, Front Door
- Observability: Azure Monitor, Application Insights, Log Analytics

**Design principles to emphasize:**
- Enterprise-grade security and compliance (law firm context — data residency, audit logging, access control)
- High availability and disaster recovery
- Structured, top-down presentation: requirements → HLD → components → tradeoffs
