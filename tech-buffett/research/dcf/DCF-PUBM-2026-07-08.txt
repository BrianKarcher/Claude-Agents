2. THE UNDERWRITING COLUMN: SBC-ADJUSTED FCF (marching order #2)
--------------------------------------------------------------------------------
Per the deep dive, valuation is underwritten on SBC-ADJUSTED FCF, not reported
FCF. FY2025 baseline bridge (from filing extract):

  +---------------------------------+---------+---------+
  | FY2025 owner-earnings bridge    | $M      | % rev   |
  +---------------------------------+---------+---------+
  | Net cash from operations (OCF)  |  81.1   | 28.7%   |
  | (-) PP&E capex                  | (14.3)  |         |
  | (-) Capitalized software        | (20.5)  |         |
  | = Reported FCF                  |  46.3   | 16.4%   |
  | (-) Stock-based comp (real cost)| (38.4)  | 13.6%   |
  | = SBC-ADJUSTED FCF              |  ~7.9   |  2.8%   |
  +---------------------------------+---------+---------+

This ~$7.9M (2.8% margin) is the TRUE trailing owner-earnings figure. At EV ~$484M
(mkt cap $629M - net cash $145M) that is ~61x trailing SBC-adj FCF at the current
price — expensive on a static basis. The entire case is the RECOVERY curve.

Model mechanics for SBC-adj FCF each year:
  SBC-adj FCF = Adj EBITDA
              - SBC (real cost, modeled as % of rev declining per scenario)
              - Capex (PP&E + capitalized software; kept as REAL cost ~$33-40M)
              - Cash taxes (18% of a normalized pre-tax proxy; PUBM's low reported
                rate reflects credits that may not persist — use 18% conservatively)
              + D&A is inside EBITDA add-back but capex replaces it as the real
                reinvestment cost, so we do NOT double-count.
Simplification: because D&A (~$44M) >> capex (~$35M), and adj EBITDA already adds
back D&A and SBC, we build SBC-adj FCF as:
  SBC-adj FCF = Adj EBITDA - SBC - Capex - Cash taxes + (D&A - capex) NWC-neutral.
For transparency the model below shows the compact form:
  SBC-adj FCF ≈ (Adj EBITDA x (1 - cash-tax drag)) - SBC - Capex_net-of-D&A-benefit.
Given D&A > capex, the (D&A - capex) term is a modest positive; we net it into a
single "reinvestment & tax" line per year to avoid clutter. All figures conservative.

6. TERMINAL VALUE — BOTH METHODS (marching order #11)
--------------------------------------------------------------------------------
METHOD A — Perpetuity growth (terminal g: BEAR 2.0% / BASE 3.0% / BULL 3.5%):
  TV_A = FY2035 SBC-adj FCF x (1 + g) / (r - g)
METHOD B — Exit multiple on FY2035 SBC-adj FCF (BEAR 7x / BASE 10x / BULL 14x),
  cross-checked vs EV/EBITDA (BEAR 6x / BASE 9x / BULL 12x).
Final TV = average of Method A and Method B.

Terminal values at the 12% PRIMARY hurdle (r=12%):

  BEAR (FY35 SBC-adj FCF -$18M): Perpetuity g=2%, r=12% => TV_A = -$18 x 1.02 /
    0.10 = -$184M (negative — a business that burns owner cash in perpetuity has
    negative going-concern value). Method B exit multiple on NEGATIVE FCF is not
    meaningful; instead we FLOOR the bear terminal at a liquidation/net-cash-plus-
    stub basis. Bear EV terminal floored at EV/EBITDA 6x x $47.9M = $287M (the
    EBITDA is positive even if owner FCF is not, because SBC is non-cash to the
    enterprise in a takeout). Bear TV = $287M (floored, not averaged with the
    negative perpetuity — see sec 8 bear note).

  BASE (FY35 SBC-adj FCF +$52M):
    Method A: $52 x 1.03 / (0.12-0.03) = $595M
    Method B: 10x x $52 = $520M  (EV/EBITDA cross-check: 9x x $168.3M = $1,515M —
      much higher; the SBC-adj-FCF multiple is the CONSERVATIVE binding one, used)
    TV_base = avg($595M, $520M) = $558M

  BULL (FY35 SBC-adj FCF +$136M):
    Method A: $136 x 1.035 / (0.12-0.035) = $1,656M
    Method B: 14x x $136 = $1,904M
    TV_bull = avg($1,656M, $1,904M) = $1,780M
